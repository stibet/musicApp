# Makam Coach – Final Source Package

Bu paket, uygulamanın kaynak kodudur.

## İçerik
- Expo / React Native kaynak kodu
- Geliştirilmiş ana ekran
- Makam Koçu ekranı
- 6 makam: Hicaz, Rast, Uşşak, Hüseyni, Nihavend, Segâh
- 4 enstrüman: Piyano, Klarnet, Keman, Ney
- Türk / Batı / Birlikte nota görünümü
- Transpoze seçenekleri
- Dizi modu + örnek makam cümlesi modu
- Adım adım doğru / yanlış geri bildirimi
- Seans özeti ve geçmiş kayıtları

## Çalıştırma
1. `npm install`
2. `npx expo start`
3. Web ya da cihaz/simülatör üzerinden aç

## Not
Mikrotonal doğruluk ve canlı pitch analizi web tarafında daha tutarlı çalışır. Native cihaz tarafı bu kaynakta ürün prototipi seviyesindedir.
